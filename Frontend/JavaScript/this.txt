Контекст выполнения функции – значение переменной this внутри функции.

Значение this – указывает на объект в контекстке которого была вызвана.
Переменная this при вызове конструктора, будет указывать на объект, унаследованный
от прототипа конструктора.

Разлечают 5 вариантов чему присваивается this:
  1. Глобальная область видимости. Когда мы используем this в глобальной области, 
     она будет просто ссылаться на глобальный объект.
     this;

  2. Вызов функции. Тут this также ссылается на глобальный объект В ES3
     и на undefined в ES5.
     foo();

  3. Вызов метода. В данном примере this ссылается на test.
     test.foo();

  4. Вызов конструктора. Если перед вызовом функции присутствует ключевое слово new,
     то данная функция будет действовать как конструктор. Внутри такой функции this
     будет указывать на новосозданный Object.

  5. Переопределение this. Когда мы используем методы call или apply, то внутри
     вызываемой функции this явным образом будет присвоено значение первого
     передаваемого параметра.
     function foo(a, b, c) {}

     var bar = {};
     foo.apply(bar, [1, 2, 3]); // массив развернётся в a = 1, b = 2, c = 3
     foo.call(bar, 1, 2, 3); // аналогично
