Работа такого кода:

for (let index = 1; index <= 10; index++) {
  setTimeout(() => {
    console.log(index);
  }, 2000);
}

отличается тем, что при использовании var в консолько через 2 секунды
выведится 10 раз число 10, потому что var будет вытянута из цикла
из-за хостинга, и в глобальной области видимости она станет 10,
после этого запустится 10 сетТаймаутов, со значением 10.

при использовании let, при каждой иттерации, будет создаваться своя
переменная let, поэтому вывод будет 1..10.
