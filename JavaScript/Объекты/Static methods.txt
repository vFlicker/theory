Обычно статические методы используются для реализации функций, принадлежащих классу, но не к
каким-то конкретным его объектам.

Статические свойства и методы наследуются.

Статические методы также используются в классах, относящихся к базам данных, для
поиска/сохранения/удаления вхождений в базу данных, например, предположим, что
Article - это специальный класс для управления статьями статический метод для
удаления статьи:

Article.remove({id: 12345})

Ещё одно приминение статических методов:

class Article {
  constructor(title, date) {
    this.title = title;
    this.date = date;
  }

  static compare(articleA, articleB) {
    return articleA.date - articleB.date;
  }
}

let articles = [
  new Article("HTML", new Date(2019, 1, 1)),
  new Article("CSS", new Date(2019, 0, 1)),
  new Article("JavaScript", new Date(2019, 11, 1))
];

articles.sort(Article.compare);

console.log(articles[0].title);
