var — область видимости переменной ограничивается функцией. Имеет hoisting
(при создании переменная инициализируется со значение undefined).

let — область видимости переменной ограничивается блоком. Не имеет hoisting.
Переменную можно переназначить.

const — область видимости переменной ограничивается блоком. Не имеет hoisting.
Переменную нельзя переопределить.
