## Перерахування, словник, константа

### Enum (перерахування)

Набір однотипних констант, що виносяться на початок модуля

```ts
const StatusCode = {
    OK: 200,
    NOT_FOUND: 404,
    BAD_REQUEST: 400,
};

enum StatusCode {
    Ok = 200,
    NotFound = 404,
    BadRequest = 400,
}
```

### Словник

Об'єкт де щось зберігається

```js
const valueToAnotherValue = {
    one: "class-one",
    two: "class-two",
    three: "class-three",
};
```

### Об'єкт-константа

```js
const BLANK = {
    description: "",
    task: "",
    isArchive: false,
    isFavorite: false,
};
```
